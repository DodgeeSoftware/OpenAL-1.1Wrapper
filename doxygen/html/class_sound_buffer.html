Detailed Description

A container around an OpenAL AudioBuffer AudioBuffers are PCM Data and this wrapper uses libsndfile to extract our PCM Data and create an OpenAL AudioBuffer. We Pass this Object to a Sound, Sound2D or Sound3D class so that we can play it.
