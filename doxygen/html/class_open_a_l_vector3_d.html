Detailed Description

A simple class for for a mathematical 3D Vector The OpenALVector3D contains an x,y,z co-ordinate pairs and overloaded operations to support vector comparison addition, subtraction. It also has standard functions for vector maginitude and dot/cross product aswell as normalisation.
