Detailed Description

A container around an OpenAL Source with positin functions and a FileStream This class will open a file stream and read it little by little creating buffers which are cycled to create seamless audio (as opposed to Sound/Sound2D/Sound3D which have to load SoundBuffers ) All formats supported by libsnd are supported. I deliberatly don not support mp3 because it is a proprietary format requiring royalites use ogg files instead.
